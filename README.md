# 🤖 Agentic-Trade-Bot

[![Agentic AI](https://img.shields.io/badge/AI-Agentic%20Workflow-red.svg)](https://github.com/TanNgo1/Agentic-Trade-Bot)

An autonomous AI agent designed to analyze market sentiment, technical indicators, and execute trades based on LLM-driven decision logic.

## 🌟 Features
- **Sentiment Analysis:** Scrapes news and social media to gauge market mood.
- **Strategic Execution:** Combines RSI, MACD, and LLM insights for entry/exit points.
- **Risk Management:** Built-in stop-loss and position sizing logic.

## 🛠️ Architecture
- **Brain:** OpenAI GPT-4o / Claude 3.5 Sonnet
- **Data Engine:** Yahoo Finance & Crypto APIs
- **Logic:** LangChain / AutoGPT style workflows
